# Demo API Wiki

> Важно: данная документация является **обезличенной демонстрационной версией** пользовательской wiki для внешних интеграций. Она подготовлена в учебных целях на основе типовых сценариев коммерческой системы, но **не содержит реальных URL, токенов, идентификаторов, схем хранения данных и внутренней бизнес-логики**.

## Назначение

Эта wiki показывает, как может быть оформлена документация внешнего API для подключения сторонних систем к сервису управления заявками и организациями.

Документация предназначена для:

- разработчиков интеграций;
- аналитиков;
- технических специалистов заказчика;
- команд, подключающих систему к собственным сервисам.

## Что описано в wiki

- общая модель работы API;
- аутентификация;
- работа с организациями;
- работа с локациями;
- создание и чтение заявок;
- сообщения по заявкам;
- webhook-события;
- обработка ошибок;
- примеры запросов и ответов.

## Ограничения демонстрационной версии

В этой версии специально заменены:

- реальные домены на `api.example.local`;
- реальные токены на `demo_token_xxx`;
- реальные идентификаторы на тестовые значения;
- реальные бизнес-правила на упрощённые примеры;
- внутренние названия сущностей на безопасные публичные аналоги.

## Базовый адрес API

Во всех примерах ниже используется демонстрационный адрес:

```text
https://api.example.local/v1
```

## Формат обмена

Обмен данными выполняется в формате JSON.

Пример заголовков:

```http
Content-Type: application/json
Accept: application/json
Authorization: Bearer demo_token_123456
```
