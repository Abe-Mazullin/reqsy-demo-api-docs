# Sanitized Demo API Wiki

Эта папка содержит обезличенную учебную wiki-документацию для открытой публикации.

## Назначение

Материалы подходят для:

- публикации в открытом GitHub-репозитории;
- демонстрации формата документации интеграционного API;
- включения в отчёт по учебной работе;
- показа структуры пользовательской технической wiki без раскрытия коммерческих данных.

## Что здесь важно

- все URL демонстрационные;
- все токены тестовые;
- все идентификаторы условные;
- структура и стиль соответствуют реальной API-документации;
- содержимое не раскрывает внутреннюю реализацию коммерческого продукта.
