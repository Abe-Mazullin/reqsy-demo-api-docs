# ReqSy Demo User Wiki

This repository is used as a public entry point to the demo user documentation for **ReqSy**.

The main documentation is published in the **Wiki** section:

- [Open GitHub Wiki](https://github.com/Abe-Mazullin/reqsy-demo-api-docs/wiki)

## What you can find there

- user-facing instructions for the main system features;
- pages about login, user cabinet, organizations, locations, tickets, support and notifications;
- screenshots of the demo interface prepared for documentation purposes;
- sanitized educational materials safe for public sharing.

## Important note

The public documentation is demonstrational and does not include commercial-sensitive data, internal configuration or private project materials.
